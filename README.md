# Automated Resume Parser

## Features

- Upload Resume PDF
- Extract Name
- Extract Email
- Extract Phone
- Extract Skills
- Extract Education
- SQLite Database
- Search Resume
- Delete Resume
- Resume Dashboard
- Export CSV

## Technologies

- Python
- Flask
- SQLite
- pdfplumber
- HTML
- CSS
- JavaScript
- Pandas

## Run

pip install -r requirements.txt

python app.py
